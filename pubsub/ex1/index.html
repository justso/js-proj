Option 1: Using jQuery 1.7's $.Callbacks() feature:
$.Callbacks are a multi-purpose callbacks list object which can be used as a
base layer to build new functionality including simple publish/subscribe
systems. We haven't yet released the API documentation for this feature just
yet, but for more information on it (including lots of examples), see my post
on $.Callbacks() here:
http://addyosmani.com/blog/jquery-1-7s-callbacks-feature-demystified/.
