Option 4: Third-party Plugins A number of jQuery plugins have been written
which provide a pub/sub implementation free of reliance on .bind()/.trigger()
(and consequently .on()/.off). Should you wish to use one of these solutions,
I'm happy to recommend Peter Higgin's plugin available here:
https://github.com/phiggins42/bloody-jquery-plugins/blob/master/pubsub.js.

Demo: http://jsfiddle.net/zkwra/

have been written (Shameless plug: I've also done one
